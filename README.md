## Description

This is a Nest.js project for global error exception handling with Rollbar.

## Before you start

Create a `.env` file in the root directory of the project. Add the following line:

```
ROLLBAR_ACCESS_TOKEN={YOUR ROLLBAR ACCESS TOKEN}
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
